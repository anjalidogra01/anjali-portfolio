Household Services Web App

📌 Project Overview

A full-featured multi-user web application that allows customers to book household services such as plumbing, electrical, and cleaning. The platform supports roles like Customer, Professional, and Admin.

🎯 Objective

To digitize and streamline the service request process for household needs, enabling easier booking, tracking, and feedback through a modern web interface.

🛠️ Tech Stack

  • Frontend: Bootstrap 5, HTML, CSS
  • Backend: Flask (Python), Jinja2 templates
  • Database: SQLite
  • Version Control: Git, GitHub

🚀 Features

  • User Registration and Login (Customer & Professional)
  • Admin dashboard to approve/reject professionals
  • Search services by location, pincode, or category
  • Request tracking with status updates (Requested, Accepted, Closed)
  • Customer feedback and professional rating system
